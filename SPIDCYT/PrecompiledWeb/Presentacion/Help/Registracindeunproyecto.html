Alta de Proyecto

El alta consiste en llenar los campos necesarios para poder registrar un nuevo proyecto en el sistema.

En negrita  se representan los datos que son obligatorios.

  1. Llene los datos del proyecto.
    1. Incluya si es un proyecto es categorizado, seleccionando el checkbox que habilitará el campo de Número de Referencia.
    2. Seleccione el Director, en caso de necesitar Asesor Cientifico el combo se llenará con las opciones
    3. Tenga en cuenta que si quiere dar de alta el director, deberá seleccionar el checkbox.
  2. Presione Dar de alta y continuar para ir a la siguiente pantalla.
    1. Solo se accederá si los datos son correctos. En caso de no serlos será avisado para poder corregirlos.

3. Ingresar integrantes al proyecto:

    1. Ingresar el rol.
    2. Seleccionar el investigador/becario.
    3. Seleccionar la fecha de alta (actual por defecto).
    4. Presionar agregar (se mostrarán en la tabla inferior).
    5. Presionar finalizar para conservar los cambios.

4. Será redirigido a la administración de proyectos.


*Estas mismas pantallas son utilizadas para la modificación de los datos
